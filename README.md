# 함수

- `pushToReceiver` : 친구 추가를 보냈을 때 해당 친구에게 푸시 알림을 전송합니다.
- `open_session` : 운전자가 세션을 열었을 때, 크루원들에게 푸시 알림을 전송합니다.
